# ASC Frontend

Next.js (App Router) + TypeScript + Tailwind CSS + Material UI + AG Grid Enterprise.

See the [repo-root README](../README.md) for the full setup (backend, database, running both sides together) and the current feature roadmap.

Quick start (backend must already be running — see root README):

```bash
npm install
cp .env.local.example .env.local
npm run dev
```
