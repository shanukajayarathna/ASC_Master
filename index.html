Exports

Every table, chart and report — exportable, wherever you're working.

CSV, Excel and PDF export are live today

Use "Export" in Catalogue Manager's toolbar for CSV/Excel of the current filtered view, or select lots and use "Export selected" from the bulk toolbar. In Reports, "Print / Save as PDF" exports the generated report as a PDF via your browser's print dialog, and "Export data (Excel)" pulls out the underlying table. Standalone PNG/JPEG export for individual charts is still on the roadmap.
